# herramientas usadas
Frontend: React + Vite + Tailwind CSS
Backend: Node.js + Express
Control de versiones: Git & GitHub

# como hacer funcionar la app
1) Clonar el repositorio https://github.com/mar-ilyn/tasks-app.git
2) Inicializar backend
   cd backend
   npm install
   npm run dev
3) Inicializar frontend
   cd frontend
   npm install
   npm run dev
